Contribute to OpenArcade

Home Host Play Contribute

Submit a game

Create a new folder in games/ with a manifest.json and a client/ folder containing index.html, style.css, and main.js.

Manifest requirements

Each manifest must include id, name, version, description, author, and wip.

Open a PR

Fork the repo, commit your changes, and open a pull request. Include screenshots if your game has UI.
