What is OpenArcade?

OpenArcade is a GitHub-first collection of party-style games. A local hub hosts the games and lets the host activate a title for players.

Quick start

Clone the repo and run the hub:

cd hub then npm install and npm start.

Open source

Games live in games/ as simple plugins. Everyone can contribute new games via pull requests.
